# PHP-noteboard
>使用php语言制作的留言板
>数据库文件在根目录下php_noteboard.sql
功能上有：  
  1添加留言  
  2删除留言（验证密码）  
  3编辑留言（验证密码）  
  4选择留言墙纸色  
  5上下页切换
